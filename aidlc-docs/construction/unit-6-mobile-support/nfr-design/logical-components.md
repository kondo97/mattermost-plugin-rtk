## Component 2: Sender

**File**: `server/push/push.go`
**Package**: `push`

```go
type Sender struct {
    api plugin.API
}

func NewSender(api plugin.API) *Sender
func (s *Sender) SendIncomingCall(session *kvstore.CallSession) error
func (s *Sender) SendCallEnded(session *kvstore.CallSession) error
```

**Internal helper** (shared by both methods):
```go
func (s *Sender) sendToMembers(
    session *kvstore.CallSession,
    notifType string,
    subType string,
) error
```

Both `SendIncomingCall` and `SendCallEnded` delegate to `sendToMembers`
with different `notifType` / `subType` values, avoiding code duplication.

**Mattermost API calls made by Sender:**

| Call | Purpose |
|---|---|
| `api.GetChannel(channelID)` | Fetch channel type, TeamId, DisplayName |
| `api.GetUser(creatorID)` | Fetch caller Username |
| `api.GetChannelMembers(channelID, 0, 8)` | Fetch up to 8 recipients |
| `api.SendPushNotification(notification, userID)` | Dispatch to push proxy |
